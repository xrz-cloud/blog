可能出现的意外及解决方法（建议看完全部再开始）

2020-07-26 1 min read # 技术 # XTC # 刷机

开机后掉root了

方法1 方法2

方法1

1、输入adb reboot recovery进入rec

2、点击 Install

3、（需提前把Magisk-20.4(20400).zip导入/sdcard/）点击Install Zip

4、在列表中找到magisk的zip，点击滑动刷入，跑完之后，重启手表，成功

方法2

1、输入adb reboot recovery进入rec

2、点击 Install

3、（需连接手表）点击ADB Sideload

4、滑动开启此模式
5、连接手表，打开搞机助手[下载]
6、连接成功，点击 恢复模式

7、点击Sideload 刷机模式

8、选择刷机包，Magisk-20.4(20400).zip，开始刷机，跑完之后，重启手表，成功
