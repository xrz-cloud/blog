可能出现的意外及解决方法（建议看完全部再开始）

2020-07-26 1 min read # 技术 # XTC # 刷机

开机后掉root了
1、输入adb reboot recovery进入rec

2、点击 Install

接下来有两种方法
方法1
3-1-1、（需提前把magisk[下载]导入/sdcard/）点击Install Zip

3-1-2、在列表中找到magisk的zip，点击滑动刷入，跑完之后，重启手表，成功

方法2
3-2-1、（需连接手表）点击ADB Sideload

3-2-2、滑动开启此模式
3-2-3、连接手表，打开搞机助手[下载]
3-2-4、连接成功，点击 恢复模式

3-2-5、点击Sideload 刷机模式

3-2-6、选择刷机包，magisk的zip[下载]，开始刷机，跑完之后，重启手表，成功
